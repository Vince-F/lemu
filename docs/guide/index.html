# What is LEMU?

Lemu is a desktop application to configure and run your BackstopJS. It aims to provide a simple and guided experience towards building your test scenario for people more accustomed to GUI than configuration files and command line interface.

You can download Lemu on the official GitHub repository in the releases section. Builds are available for Windows, Linux and MacOS.

To know how to start using Lemu jump into the Getting sarted section.
